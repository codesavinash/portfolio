# Images Directory

## Logo Files

### Portfolio Logo
- `logo.png` - Main portfolio logo for the navigation bar
  - Recommended size: 40-50px height (width will scale proportionally)
  - Format: PNG with transparent background recommended
  - The logo will appear next to "Portfolio" text in the navigation bar

## Profile Picture
- `profile.jpg` - Your profile picture for the About section
  - Recommended size: 400x400px to 600x600px (square format)
  - Format: JPG or PNG
  - The image will be displayed in a circular/square frame with rounded corners
  - If the image is missing, a placeholder will be shown

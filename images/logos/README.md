# Logo Images Directory

Place your skill logo PNG files in this directory with the following names:

## Design Tools
- `figma.png` - Figma logo
- `photoshop.png` - Adobe Photoshop logo
- `adobe-xd.png` - Adobe XD logo
- `adobe-illustrator.png` - Adobe Illustrator logo

## Development
- `html.png` - HTML logo
- `css.png` - CSS logo
- `javascript.png` - JavaScript logo

## Certifications
- `aws.png` - AWS logo
- `comptia.png` - CompTIA logo
- `lenovo.png` - Lenovo logo

## Image Requirements
- Format: PNG (with transparent background recommended)
- Skill logos: 60x60px to 120x120px recommended
- Certification logos: 80x80px to 120x120px recommended
- The images will be automatically resized to fit their containers

## Note
If a logo file is missing:
- Skill cards will automatically fall back to displaying a text abbreviation (F, PS, XD, AI, HTML, CSS, JS)
- Certification cards will fall back to displaying text (AWS, CompTIA, Lenovo)

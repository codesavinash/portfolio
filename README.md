# Portfolio Website

A modern, minimalist portfolio website built with vanilla HTML, CSS, and JavaScript.

live demo https://codesavinash-github-io-sau7.vercel.app/

## Features

- **Responsive Design**: Fully responsive layout that works on mobile, tablet, and desktop devices
- **Modern UI**: Clean, minimalist design with smooth animations and transitions
- **Smooth Scrolling**: Navigation with smooth scroll behavior
- **Mobile Menu**: Hamburger menu for mobile devices
- **Contact Form**: Functional contact form with validation
- **Sections**: 
  - Hero/Introduction
  - About Me
  - Skills (Design Tools & Development)
  - Experience
  - Projects (Placeholder cards)
  - Contact

## Technologies Used

- HTML5
- CSS3 (with CSS Variables, Flexbox, Grid)
- Vanilla JavaScript
- Google Fonts (Inter)

## File Structure

```
/
├── index.html          # Main HTML structure
├── css/

├── js/

```

## Getting Started

1. Clone or download this repository
2. Open `index.html` in your web browser
3. No build process or dependencies required!

## Customization

### Colors
Edit the CSS variables in `css/style.css`:
```css
:root {
    --primary-color: #2563eb;
    --primary-dark: #1e40af;
    /* ... other variables */
}
```

### Content
- Update personal information in `index.html`
- Replace placeholder email in the contact section
- Add your projects to the projects section
- Update social media links

### Styling
All styles are in `css/style.css`. The design uses a mobile-first approach with responsive breakpoints at:
- Mobile: 480px
- Tablet: 768px
- Desktop: 1024px
